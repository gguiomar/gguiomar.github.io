Research

My research aims at uncovering the processes by which organisms learn to represent the world and the fundamental mathematical structures that support them. Trained as a physicist, I use a variety of tools and frameworks inherited from neuroscience, mathematics and machine learning to try to answer these questions whilst at the same time striving for biological realism and expositional simplicity. Besides this I'm also interested in how we can represent these intuitions using different mediums, such as installations and music pieces developed in collaboration with various artists.

Selected Publications

"Representation learning over latent state policies" - Gonçalo Guiomar, Joseph J. Paton, Daniel McNamee (In Progress)

This project aims at understanding how acting in the world leads to a change in perception and how this could be adapted to a Reinforcement Learning framework based on Variational Inference.

"Action suppression reveals opponent parallel control via striatal circuits" - Bruno F. Cruz, Gonçalo Guiomar, Sofia Soares, Asma Motiwala, Christian K. Machens, Joseph J. Paton (2022). Nature, 607, 521-526

Built a multi-agent Reinforcement Learning model that is able to reproduce behavioral and neurophysiological data of mice in a time interval discrimination task.

"Astrophysical Constraints on the Bumblebee Model" - Gonçalo Guiomar, Jorge Páramos (2014). Phys. Rev. D, D 90, 082002

Was able to derive the strongest constraints for the existence of Dark Matter in the Sun using a symmetry breaking perturbation of Einstein's field equations.
